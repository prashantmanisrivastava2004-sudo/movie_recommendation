# 🎬 Movie Recommendation System

A Machine Learning-based Movie Recommendation System that recommends similar movies based on content similarity using NLP techniques and Cosine Similarity.

## 🚀 Features

- Search movies by title
- Get Top 5 similar movie recommendations
- Content-Based Filtering
- Machine Learning powered recommendations
- Flask web application
- Responsive user interface

## 🛠️ Technologies Used

- Python
- Flask
- Pandas
- NumPy
- Scikit-Learn
- HTML
- CSS

## 🧠 Machine Learning Approach

This project uses a **Content-Based Recommendation System**.

### Workflow

1. Collect movie dataset
2. Combine movie features (genres, keywords, cast, crew, overview)
3. Convert text data into numerical vectors using CountVectorizer
4. Calculate similarity using Cosine Similarity
5. Recommend the most similar movies
